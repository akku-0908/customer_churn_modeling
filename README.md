# Customer Churn Modeling 📊👥

This repository contains an end-to-end machine learning project designed to predict customer churn using the Telco Customer Churn dataset. It features an interactive web application built with Streamlit, allowing users to analyze churn factors and predict whether a customer is likely to leave the service.

---

## 🚀 Features

* **Interactive Web App:** A clean, user-friendly UI built with Streamlit for real-time churn predictions.
* **Predictive Modeling:** Utilizes a pre-trained machine learning model (`model.pkl`) trained on historical telecom data.
* **Data-Driven Insights:** Analyzes key customer attributes such as contract type, monthly charges, tenure, and internet service type to assess risk.

---

## 📁 Repository Structure

* `streamlit_app.py` - The main entry point for the Streamlit dashboard application.
* `2_Predict_Churn.py` - Script handling the churn prediction logic and backend processes.
* `model.pkl` - The trained serialised machine learning model.
* `Telco-Customer-Churn.csv` - The historical dataset used for analysis and training.
* `requirements.txt` - List of Python dependencies required to run the project.
